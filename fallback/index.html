Tentang ELSA

ELSA (Education Learning Smart App) adalah aplikasi PWA (Progressive Web App) yang dirancang untuk membantu Anda belajar dengan lebih efisien. Aplikasi ini dapat bekerja secara offline dan memberikan pengalaman belajar yang cepat dan responsif.

URL Aplikasi: https://shadowwd117.github.io/ELSA/

Pengembang: Shadow

Pertanyaan Umum (FAQ)

Apa itu ELSA?

ELSA adalah aplikasi pembelajaran berbasis web progresif (PWA) yang memungkinkan Anda mengakses materi belajar secara offline. Aplikasi ini dirancang untuk memberikan pengalaman belajar yang cepat dan efisien dengan antarmuka yang sederhana.

Bagaimana cara menggunakan ELSA secara offline?

ELSA menggunakan teknologi Service Worker untuk menyimpan konten secara lokal di perangkat Anda. Setelah Anda mengunjungi situs ELSA untuk pertama kali, aplikasi akan meng-cache konten penting sehingga dapat diakses tanpa koneksi internet.

Pastikan Anda telah mengizinkan penyimpanan cache di browser Anda untuk pengalaman offline yang optimal.

Bagaimana cara mengelola cache di ELSA?

Anda dapat mengelola cache melalui halaman Pengaturan:

  1. Buka halaman Pengaturan
  2. Pilih opsi "Manajemen Penyimpanan"
  3. Anda dapat melihat daftar cache dan menghapusnya jika diperlukan

Menghapus cache akan membebaskan ruang penyimpanan tetapi juga menghapus data yang memungkinkan aplikasi bekerja offline.

Apakah data profil saya aman?

Semua data profil (nama, email, foto, dll.) disimpan secara lokal di perangkat Anda menggunakan localStorage browser. Data ini tidak dikirim ke server mana pun dan hanya dapat diakses oleh aplikasi ELSA di browser Anda.

Anda dapat menghapus semua data lokal kapan saja melalui opsi "Hapus Data Lokal" di halaman Profil.

Bagaimana cara memperbarui aplikasi ELSA?

Karena ELSA adalah PWA, pembaruan dilakukan secara otomatis ketika ada versi baru. Service Worker akan mendeteksi perubahan dan memperbarui cache dengan konten terbaru.

Anda juga dapat memaksa pembaruan dengan:

  1. Menghapus cache melalui halaman Pengaturan
  2. Muat ulang halaman ELSA
Apakah ELSA mendukung perangkat mobile?

Ya, ELSA dirancang dengan pendekatan mobile-first dan responsif. Aplikasi ini dapat diinstal di perangkat mobile seperti smartphone dan tablet, serta berfungsi dengan baik di berbagai ukuran layar.

Untuk pengalaman terbaik, disarankan menggunakan browser terbaru seperti Chrome, Firefox, atau Safari.

Apa fungsi username dan email pada bagian profile?

Username dan email pada bagian profile bertujuan untuk memudahkan pengguna dalam menghubungi pengembang. Dengan menggunakan username serta email dari bagian profile, pengguna tidak perlu lagi memasukka username dan email di bagian bantuan dan FAQ

Fitur ini aman dan bertujuan untuk meningkatkan kesan terbaik untuk pengguna.
